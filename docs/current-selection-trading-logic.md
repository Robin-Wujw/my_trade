# 当前选股与买卖规则（中文伪代码版）

这份文档只写“人能看懂的规则”。程序里的英文变量名、信号名、字段名不作为解释口径。

均胜电子、多氟多、天赐材料只用于校准买卖思想，不用于反推选股模型必须选中它们。

## 一、选股逻辑

### 1. 先做硬过滤

```text
如果 股票财务质量分 < 70：
    剔除，不交易

如果 扣非增长 < 10%：
    剔除，不交易

如果 总市值 < 100亿元：
    剔除，不交易

如果 股票没有行情或基础数据不完整：
    剔除，不交易
```

### 2. 再判断股票属于哪一类候选

```text
如果 股票接近价值线，财务质量合格，价格没有明显高估：
    归为“左侧价值候选”

如果 股票属于新鲜主线板块，财务质量合格，交易基础合格：
    归为“主线候选”

如果 股票多周期走势强，财务质量合格，强度分和交易基础分合格：
    归为“强势成长候选”
```

当前更偏向右侧趋势交易，所以右侧新仓只优先做：

```text
如果 股票同时是“主线候选”和“强势成长候选”：
    允许进入右侧执行池
否则：
    不作为右侧新仓首选
```

价值线股票的处理：

```text
如果 当前是右侧市场，并且股票只是价值线逻辑：
    不开右侧新仓

如果 已经有左侧价值仓，且价格脱离价值线并出现合格右侧买点：
    可以左侧切换为右侧管理
```

### 3. 候选池排序

```text
每天先把所有候选统一打分
先保留价值和主线的核心名额
再用强势成长候选补足
最多进入正式交易池10只

没有进前10的股票：
    只能作为诊断记录
    不能因为后来涨了就倒推为金标准
```

### 4. 三个作者案例的选股口径

```text
均胜电子、多氟多、天赐材料：
    可以强制放进“买卖验证池”
    只测试买卖点是否符合白大思想
    不要求正式选股模型必须选中
    也不能因为这三只股票反过来放宽选股条件
```

## 二、买入逻辑

### 1. 买入前总检查

```text
如果 股票不在候选池：
    不买

如果 股票没有合格技术买点：
    不买

如果 技术证据分不足：
    不买

如果 行业或主题已经满额：
    不买

如果 总持仓数量已满，且没有弱票可换出：
    不买

如果 单只股票仓位会超过上限：
    缩小买入；缩小后仍不合适就不买

如果 总仓位会超过100%：
    不买
```

### 2. 哪些形态可以作为右侧首仓

右侧首仓优先看“确认型买点”，不是看便宜。

```text
如果 出现 W底颈线突破：
    可以作为右侧首仓

如果 出现跳空长阳突破重要均线或结构：
    可以作为右侧首仓

如果 出现窄幅平台突破：
    可以作为右侧首仓

如果 回调波段重新向上突破一半位：
    可以作为右侧首仓

如果 上涨波段一半位跌破后重新收复：
    可以作为右侧首仓

如果 连阳上涨后一半拉回，且结构、量能、均线配合：
    可以作为右侧首仓

如果 只是上涨波段比例位支撑拉回：
    默认不能作为首仓
```

比例位支撑拉回的规则：

```text
如果 没有右侧底仓：
    支撑拉回不是买点，只记录观察

如果 已经有右侧仓位，并且这笔仓位已经证明自己：
    支撑拉回可以作为加仓点

如果 人工明确打开“拉回试错模式”：
    支撑拉回可以小仓试错
否则：
    不买
```

这条是核心：白大思想里，比例位只是支撑或观察点，不是天然买点。

### 3. 首仓仓位

```text
如果 是强确认突破买点：
    首仓约30%

如果 当前市场连续三日下行：
    首仓降到约20%

如果 是普通试错买点：
    首仓约20%

如果 人工打开拉回试错模式：
    首仓约15%

如果 是左侧价值仓切换右侧：
    新增右侧仓位不能超过原左侧仓位的一半

任何情况下：
    单只股票总暴露不超过约62.5%
```

### 4. 加仓规则

```text
如果 已有右侧仓位还没有浮盈证明：
    不加仓

如果 新买点是拉回类买点：
    加仓不超过已有右侧仓位的一半

如果 新买点是突破新高或新结构：
    加仓不超过已有右侧仓位的三分之一

如果 市场连续三日下行：
    加仓更谨慎；没有足够浮盈不加
```

### 5. 左侧价值买入

```text
如果 当前是右侧市场：
    不新增左侧价值计划

如果 股票是价值候选，价格在价值线附近，财务没有证伪：
    可以按网格小仓左侧买入

左侧每格约2%
价格每下跌约5%增加一格
左侧总仓不超过约20%

如果 价值逻辑被财务证伪：
    停止左侧计划，并清掉左侧仓
```

## 三、卖出逻辑

卖出分为止损、时间退出、分仓止盈、尾仓退出、弱票换出、价值证伪退出。

### 1. 硬止损

```text
如果 跌破入场结构止损：
    对应批次退出

如果 是尚未证明的新买批次：
    只按自己的入场止损管理

如果 是普通右侧仓：
    即使曾经盈利过，只要没变成高收益尾仓，仍受原结构止损约束
```

### 2. 突破次日确认

```text
如果 买点要求次日确认：
    次日开盘仍站在确认价上方：
        继续持有
    次日开盘没有站上确认价：
        退出该批次
```

### 3. 时间止损

```text
如果 买入后经过规定天数，最大浮盈仍不到10%：
    退出该批次

一般环境：
    给更长观察时间

弱环境、背离环境、试错仓：
    给更短观察时间
```

白大含义：买入后迟迟不证明自己，就不是好买点。

### 4. 分仓止盈

当前默认不是五份机械止盈，而是两份管理：

```text
如果 最大浮盈不到10%：
    不止盈，只看止损和时间

如果 最大浮盈达到10%但不到20%：
    只看“最大浮盈回撤一半”
    触发时卖出保护仓位

如果 最大浮盈超过20%：
    开始看多种保护信号
```

超过20%后的保护信号：

```text
如果 跌破利润底线：
    卖出一份

如果 从最高点回撤超过约10%：
    卖出一份

如果 出现顶背离，并且连续多日没有补量修复：
    卖出一份

如果 跌破最近有效量价节点：
    卖出一份

如果 上扬均线有效跌破：
    卖出一份
```

卖出顺序不是为了凑次数，而是每一个卖点必须有独立理由。

### 5. 最后一份尾仓

```text
如果 已经减到最后一份，并且当前浮盈仍很高：
    这份变成趋势尾仓
    不再因为普通容量约束被迫卖出

如果 最后一份从最大浮盈回撤一半：
    清掉尾仓

如果 最后一份当前浮盈不够高：
    仍然按普通仓位管理
    不能假装成高收益尾仓
```

当前把止盈份数调成两份，是为了避免强趋势股票太早卖碎。白大思想不是机械卖五次，而是先让强票证明，再分阶段保护利润。

### 6. 弱票换出

```text
如果 新候选出现合格买点，但持仓名额已满：
    检查已有持仓中是否有弱票

如果 某只旧票：
    持有时间够久
    没有足够浮盈
    或跌破关键均线/结构
    且新票明显更强
那么：
    安排旧票退出，给新票腾位置

如果 没有弱票可换：
    新票不买
```

### 7. 落选退出

```text
如果 股票从候选池落选：
    不立刻卖掉所有右侧仓

如果 只剩很小的右侧尾仓，并且不再符合继续持有条件：
    次日退出

如果 仍是强趋势尾仓：
    按尾仓规则继续管理
```

### 8. 左侧价值证伪退出

```text
如果 左侧股票财务质量变差：
    清掉左侧仓

如果 扣非增长不达标：
    清掉左侧仓

如果 市值或流动性硬门槛不达标：
    清掉左侧仓

如果 价值线逻辑失效：
    清掉左侧仓

如果 只是没有进入当天前10，但价值逻辑没有证伪：
    不清仓，只停止新增
```

## 四、三个作者案例怎么用

### 1. 总原则

```text
三个案例可以强制放进验证池
但只能验证买卖逻辑
不能验证选股逻辑
不能倒推选股门槛
不能为了让它们成交而放宽比例位首仓
```

### 2. 均胜电子

```text
先找白大锚点：
    上涨波段一半位约26.64
    回调波段一半位约31.33或30.18

如果 程序自动锚点和白大锚点不一致：
    记录锚点不一致
    不强行生成买点

如果 只是碰到上涨波段比例位：
    不买

如果 重新收复一半位、突破回调一半位、量价节点确认：
    才考虑右侧买入
```

### 3. 多氟多

```text
先看关键区：
    31.06附近
    29.62量价节点
    季线低价低量扣抵
    回调超过13日后的回调一半位

如果 只是上涨波段七五位或六二五位支撑：
    不开首仓

如果 节点、扣抵、时间、突破或收复同时支持：
    才考虑右侧买入

如果 已有仓位并且浮盈证明：
    支撑拉回可以考虑加仓
```

### 4. 天赐材料

```text
先分清级别：
    大级别一半位约38.40
    局部一半位约50.645
    局部七五位约57.8125

如果 只是局部比例位支撑：
    只能作为观察或已有仓加仓参考
    不能证明大趋势改变

如果 大级别结构没有恢复：
    不因为局部比例位开首仓

如果 右侧批次买入后很快触发空间止损或时间止损：
    退出该批次
```

## 五、当前结论

```text
当前选股：
    更偏向主线加成长重叠
    排除右侧市场中的价值线拖累

当前买入：
    不把比例位当首仓买点
    首仓偏向突破、收复、节点、量价确认

当前卖出：
    不机械五份止盈
    用两份管理保留更大趋势尾仓
    失败批次及时止损或时间退出

当前案例校准：
    方向符合白大
    但还没完整程序化三个案例的显式锚点
    下一步要补“锚点不一致”输出和白大关键价识别
```
