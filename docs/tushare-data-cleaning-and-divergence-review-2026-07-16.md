## 背离指标规则

### 高低价 KD

使用观察日及以前的数据：

```text
收盘RSV = (收盘价 - 9日最低价) / (9日最高价 - 9日最低价) * 100
收盘K = 通达信SMA(收盘RSV, 3, 1)
收盘D = 通达信SMA(收盘K, 3, 1)

最高价RSV = (最高价 - 9日最低价) / (9日最高价 - 9日最低价) * 100
最高价K = 昨日收盘K * 2/3 + 今日最高价RSV / 3
最高价D = 昨日收盘D * 2/3 + 今日最高价K / 3

最低价RSV = (最低价 - 9日最低价) / (9日最高价 - 9日最低价) * 100
最低价K = 昨日收盘K * 2/3 + 今日最低价RSV / 3
最低价D = 昨日收盘D * 2/3 + 今日最低价K / 3
```

### 背离判断

```text
顶背离：
价格最高价创新高，但指标强度没有创新高。

底背离：
价格最低价创新低，但指标强度没有创新低。
```

WR 是反向指标，先转成强弱值：

```text
WR强弱 = ((100 - WR10) + (100 - WR20)) / 2
```

然后再按普通指标判断顶背离和底背离。

## 使用边界

背离不能单独生成买点。

```text
底背离 + 主结构买点 + 量能改善
=> 可以提高候选排序或买点证据可信度
```

背离也不能单独清仓。

```text
最大浮盈超过20%
且 多指标顶背离
且 5日不创新高
且 量能未恢复
=> 只触发现有“背离计时”分仓止盈一份
```

强趋势里连续创新高会重置不过高计时，不能因为指标高位钝化就机械卖光。
