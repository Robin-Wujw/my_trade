# 2024-09-24 至今组合回测异常诊断

> 数据截止：本地冻结行情与候选池到 2026-07-14。
> 本文只解释诊断结论，不把任何单只股票写成金标准。

## 1. 先给结论

当前长周期回测只有约 64%，不是因为 2024-09-24 以后没有牛股，也不像是单纯为了今年收益硬调。

真正的问题是：

```text
2024-2025 年：
    大部分时间现金太多
    右侧仓位没有持续打开
    价值线里出现的强股没有顺利转成右侧趋势仓
    少数右侧买入又被次日确认或条件止损很快洗掉

2026 年：
    组合抓住了少数强趋势仓
    最终收益主要来自期末未实现浮盈
```

所以“2024-09-24 至今”和“2026 年初至今”收益接近，是因为前一段 2024-2025 基本没有赚到钱，而不是长周期应该天然只赚这么多。

未实现收益：+67.293%
最大回撤：-21.747%

```

期末主要仓位：

### 分年份暴露

    收益约 +0.522%
    平均现金约 78.13%
    平均总持仓约 21.21%

    平均总持仓约 62.78%
```

这说明 2024-2025 的主要问题不是“买错很多”，而是“基本没参与”。

## 3. 已经证伪的简单改法

### 1. 放宽主交易名额

把主交易名额从 3 只放到 5 只，并把同主题限制从 2 放到 3，长周期结果仍是：

```text
最终收益：+64.029%
```

结论：

```text
如果结果完全不变：
    说明主要瓶颈不是持仓名额太少
    而是可执行右侧买点没有形成，或者形成后很快退出
```

### 2. 打开强势拉回试错

允许领先族群的比例位支撑拉回做 15% 小仓试错后：

```text
最终收益：+67.920%
已实现收益：-6.594%
交易次数明显增加
交易成本升到 1.222%
```

结论：

```text
收益只小幅提高
但已实现亏损和交易成本变差
不能直接把支撑拉回首仓变成默认生产规则
```

### 3. 粗暴允许价值候选转右侧

把价值线候选里量价基础较强的股票临时打开右侧评估：

```text
长周期最终收益：-8.354%
```

只允许每日前 10 的价值候选转右侧：

```text
长周期最终收益：+16.176%
```

结论：

```text
2024-09-24 的确有强股被价值权限挡住
但不能简单把价值线候选全部打开右侧
否则假突破太多，组合反而被打坏
```

## 4. 强样本漏斗

这些股票不是金标准，只用来定位漏斗。

```text
中际旭创：
    2024-09-24 已在候选池
    当天分类为价值线候选
    排名第 4
    右侧权限关闭
    左侧权限打开
    但左侧名额先被其他价值候选占用

    2025-09-15 后来出现右侧买入
    仓位约 16%
    2025-10-09 被条件止损退出

新易盛：
    2025-09-23 出现右侧买入
    仓位约 27.7%
    次日因突破未确认退出

工业富联、长飞光纤、洛阳钼业：
    多数高证据信号属于比例位支撑拉回
    当前规则默认不允许它们作为普通首仓
```

这说明问题分三层：

```text
候选层：
    有些强股进入过候选池

权限层：
    价值线候选在牛市右侧转强时，右侧权限太硬

买卖层：
    真正买入后，次日确认和条件止损可能过于机械
```

## 5. 为什么不能直接追 200%

如果只看 2024-09-24 当天，候选池里确实有后来涨幅极大的股票。

但要在回测里达到 200%，通常意味着：

```text
必须在早期选中少数最大牛股
必须给到较大仓位
必须不被中途震荡洗掉
必须没有被其他假突破消耗本金和名额
```

这些要求如果用事后结果倒推，很容易变成：

```text
看到未来谁涨得最多，再回头让规则选中它
看到未来洗盘没跌死，再回头放宽止损
看到未来突破成功，再回头取消次日确认
```

这类做法不能进入正式策略。

## 6. 下一轮应该改哪里

### 改法一：建立“价值线转右侧观察”而不是“价值线直接右侧买入”

伪代码：

```text
如果 一只股票来自价值线：
    如果 它只是便宜：
        只能做左侧

    如果 它已经涨离价值线
    并且 量价基础连续走强
    并且 已经进入主流方向或强势成长观察
    并且 出现独立右侧确认买点：
        允许进入右侧观察
        但不能因为价值线身份直接买
```

关键是“先转观察，再等买点”，不是“价值候选一强就买”。

### 改法二：给强趋势失败退出增加二次确认诊断

当前逻辑：

```text
如果 买点要求次日确认：
    次日开盘仍站上确认价：
        继续持有
    次日开盘没站上确认价：
        立即退出
```

下一轮只建议做诊断实验，不直接改生产：

```text
如果 次日开盘没站上确认价：
    如果 当日收盘重新站回确认价
    并且 没有跌破原始结构止损
    并且 量能没有明显坍塌：
        记录为“盘中洗回，收盘确认”
        不在开盘直接退出
    否则：
        退出
```

这个改法的风险是会多扛失败突破，必须和保守执行口径一起回测。

### 改法三：加强汰弱留强

当前弱仓替换仍偏保守。下一轮应测：

```text
如果 已持仓：
    持有超过 5 个交易日
    当前浮盈不足 10%
    收盘跌破 20 日线
    或者 已经跌出候选池

同时 新候选：
    分数明显更高
    买点证据更强
    止损距离更窄

那么：
    次日先卖弱仓
    再给强候选名额
```

这比单纯增加持仓数量更符合“汰弱留强”。

## 7. 不能碰的红线

```text
不能把中际旭创、新易盛、工业富联等股票写入策略
不能用 2026-07-14 以后才知道的涨幅反推 2024 买点
不能把价值线候选无条件变成右侧候选
不能把比例位支撑拉回无条件变成首仓
不能为了 200% 取消止损、行业限制或总仓位限制
不能只看最终收益，不看已实现收益、未实现收益、回撤和交易成本
```

## 8. 当前判断

```text
今年至少 30%：
    当前默认口径已经满足，约 +60.123%

2024-09-24 至今至少 200%：
    当前正式口径没有达到
    简单放宽仓位、拉回首仓、价值转右侧都没有达到

下一步正确方向：
    先修“价值线转右侧观察”
    再测“突破次日确认是否过严”
    再测“汰弱留强是否释放强股名额”
```
