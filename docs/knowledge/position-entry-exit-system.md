## 总体流程

```text
基本面与产业资格
→ 判断市场阶段
→ 选择左侧或右侧
→ 寻找真实技术支撑
→ 用波段比例辅助定位
→ 每笔仓位绑定独立止损
→ 浮盈达到门槛后进入分仓止盈
→ 新高、补量或结构变化动态重置
```

分析优先级仍是：市场结构 → 量 → 价格 → 技术指标。技术指标用于描述状态和背离，不单独预测价格。

## 左侧建仓

### 适用条件

- 企业能够使用PE、PB、现金流或基本价值线可靠估值。
- 产业趋势没有明显恶化，财务逻辑仍有效。
- 股价处于基本价值线、历史估值低位或可信的长期技术支撑。
- 能判断未来可能有上涨波段，但无法确定启动时间。

### 五份建仓

左侧使用部分仓位按预设价格网格买卖。五份是一种建仓批次示例，不是强制层数，也不是五份止盈。

```text
单份仓位 = 左侧计划总仓 / 5
```

示例：计划总仓30%拆五层时每层6%，拆三层时每层10%；计划总仓25%拆五层时每层5%。每层也可以使用不同仓位，但总和不得超过左侧计划上限。价格网格必须在建仓前确定，到价执行，不因盘中情绪修改。

左侧网格允许往返交易：每层在开始前分别配置买入价、卖出价、仓位比例以及是否为核心仓；最低价触及预设买价按预设价买入，最高价触及预设卖价按预设价卖出，卖出后再次回落触及原买入价可以重新买入。同一根日K同时覆盖买卖价时不假定盘中先后顺序，不做同档日内往返。是否保留核心仓也应写入计划，不能由程序根据固定价差猜测。收盘确认适用于右侧结构信号，不套用于提前规划的网格限价单。

### 左侧退出

- 基本面、估值或产业逻辑失效。
- 买入依据的长期支撑或预设锚点失守。
- 用户买入前设定的固定幅度止损触发。

左侧默认不使用右侧的5至13日时间止损，因为左侧本来就不预期立即启动。

程序口径补充：日榜落选不是左侧清仓信号。只有候选诊断行明确给出价值/财务证伪原因，例如质量门槛失败、扣非增长失败、价值线实质下修或 `value_financial_falsification`，才在下一交易日开盘清掉左侧计划；没有失败原因时只停止新增左侧买入。

## 右侧建仓

### 持股数量纪律

- 所有策略合计持股不超过5只，日常最好控制在3只以内。
- 主交易名额默认3只，参数敏感性测试最多可放到5只。所谓“结构不明一次只做一只”限制的是新仓开启节奏，不要求卖掉原有合规持仓，也不把总持仓永久限制为1只。
- Formula33连续上行3日后：第一只右侧仓最大浮盈达到10%后才可开第二只；已有两只时，两只都达到10%后才可开第三只。
- Formula33尚未连续上行3日：第一只达到20%后才可开第二只；已有两只时，至少一只达到20%、另一只达到10%后才可开第三只。未达标的弱仓按自身止盈止损退出后，可以重新按当前名额判断新仓。
- Formula33进入右侧观察或主动阶段时，左侧价值标的最多1只；同一股票同时持有左仓和右仓也计入这个左侧名额。若右侧行情启动时已有多只左侧标的，只保留“已有右仓优先、候选分更高、持仓暴露更大”的一只，其余按组合限额在下一交易日开盘退出；这不是价值证伪，也不使用日榜落选作为理由。非右侧行情下，左侧新计划仍受全部持仓5只、主交易名额、总仓100%和单股上限约束。
- 同一族群优先只做1只、最多持有2只；程序需要可靠行业映射后才能自动执行该限制。
- 数量限制和账户总仓100%限制同时生效，先达到任一上限即停止开新股票。

### 汰弱留强

- 右侧名额占满时，不因新股票出现买点就机械卖出已有盈利趋势仓。
- 只有已有仓同时满足退出当前候选池、收盘低于MA20、持有至少5个交易日、浮盈不足10%，并且出现优先级更高的新候选买点时，才安排汰弱。
- 汰弱决定使用收盘后信息，旧仓在下一交易日开盘退出；不假设同日收盘卖出后还能回到盘中买入新股票。
- 新候选仍需后续再次满足有效条件单才买入，不为回测收益假设必然换仓成功。

### 适用条件

- 基本面无明显问题。
- 出现可定义“看错条件”的技术买点。
- 市场结构允许进行右侧试错。

### 仓位

```text
Formula33下行或尚未确认：单次右侧建仓20%
Formula33连续上行3日：标准拉回型右侧首仓30%
自动识别或显式计划的合理技术买点：33连续下行不足3日用30%，连续下行达到3日用20%
突破新高加仓：已有仓位的1/4～1/2
```

仓位大小服务于账户风险。三成仓配合10%极限止损，对账户影响约3%。

Formula33连续下行3日后降低右侧频率：空仓仍可按有效信号用20%试错；已有持仓时按上面的20%/10%逐级门槛开新仓，不要求所有持仓都达到20%。30%是33明确上行时标准拉回买点的正常首仓，不是所有右侧信号的固定仓位。

### 右侧买点类型

程序执行多类可审计右侧买点，并支持显式配置的比例结构条件单：

所有自动买点先累计证据分，生产回测的执行门槛为6分。基础分来自主结构；收盘量能确认加1分，明确上行趋势中的有效量价节点加1分，至少3个未失效节点构成节点簇再加2分，MA20/MA60/MA120扣抵共振分别加1/2/3分，大波段结构加2分。低于门槛的信号保留用于观察和排序，不下单。

1. W底颈线：使用左右各3根K线确认的两个低点，两低点误差不超过8%、间隔至少10个交易日，收盘放量突破中间颈线才触发，颈线为批次止损。
2. 整理平台突破：此前21日高低振幅不超过15%，当日收盘放量突破平台上沿才触发；单纯盘中触及21日高点不再买入。
3. 跳空长阳越过MA20/MA60：缺口、3%以上阳线实体和量能同时确认，缺口下沿为止损。
4. 有效量价节点：最近60日成交量至少为区间中位量1.5倍、当日涨幅至少2%的候选节点，形成当日不使用；次日后才能以两日最低价为支撑，在支撑上浮2%～4%范围内收盘确认。此后任一最低价跌破支撑，该历史节点永久失效，重新站回也不得复活。明确上行趋势给最近一个有效节点加1分；至少3个未失效节点组成节点簇再加2分，仍需总证据达到6分才执行。程序在事件中记录节点日期、确认日期、支撑价和当时有效节点数。
5. 连阳一半与扣抵共振：最近至少3根连阳的一半位置可作为独立拉回结构。MA20、MA60、MA120扣抵低量低价只作为既有W底、平台、缺口、波段比例、量价节点或连阳一半结构的加分证据，分别加1、2、3分，不得独立触发交易。扣抵共振要求结构价位于均线3%以内、均线上扬、被扣价格低于均线至少5%、被扣成交量低于近60日中位量的80%，且当前成交量不低于被扣量；涨幅至少50%的大波段结构另加2分。买入后仍使用原始结构而非扣抵分数止损。
6. 上涨波段比例拉回与回调波段一半突破：仍使用独立L-H-P锚点、幅度和整理天数门禁。U50跌破后收复与回调波段一半突破必须收盘站上结构位且当日成交量不低于5日/10日较高基准量，不能因盘中短暂越过就成交；买入结构就是该批次止损结构。

结构锚点及50%、62.5%、75%价位默认启用，不设“默认关闭”。显式配置锚点优先；自动锚点必须通过比例级别对应的时间窗口和MA共振过滤，且回调波段50%突破仍须满足同级上涨结构62.5%曾被跌破、幅度检查和至少13个交易日整理。可用 `--no-auto-price-structure` 仅做关闭敏感性测试。

## 价格结构比例资料

50%、62.5%、75%是默认参与提醒和回测的价格结构工具，不是单股仓位比例，也不是天然支撑。显式计划锚点优先；自动锚点只有在时间、均线等共振门禁通过后才可触发。

自动锚点不得使用固定窗口全局最低/最高。程序先识别左右各3根K线确认的拐点和放量启动日，再同时保留小级别与趋势级别L-H结构。小级别结构用于寻找技术支撑附近的比例拉回；只有通过 `U50 × 1.1 < U625` 的趋势级别结构，才可参与趋势改变和回调R50资格判断。

## 两种50%必须分开

### 上涨波段50%支撑位

设上涨波段低点为 `L_up`，高点为 `H_up`：

```text
上涨波段50%支撑位 = L_up + (H_up - L_up) × 50%
```

股价从高点向下回调至上涨波段50%支撑位，并且附近存在均线、量价节点、缺口或趋势支撑时，可以拉回试错。

```text
向下接近上涨波段50%支撑位 + 支撑有效 → 买入
收盘有效跌破上涨波段50%支撑位或绑定支撑 → 止损该批次
```

上涨波段U50有效收盘跌破后，后来重新向上突破U50，可以作为“原趋势改变被证伪”的独立右侧买点；其止损仍是收盘重新跌破U50。该信号不同于回调波段 `R50=(H+P)/2`，U75和U625不因普通跌破后重新站回而机械生成突破买点。

天赐材料案例中，小级别上涨起点约15.36元，高点约61.44元：

```text
上涨波段50%支撑位 ≈ 38.40元
```

截至2026年7月10日收盘40.06元，已接近这一支撑候选。是否买入仍需验证38.4附近存在真实支撑和量能条件。

### 回调波段50%突破位

设此前上涨高点为 `H`，之后形成的回调低点为 `L_pullback`：

```text
回调波段50%突破位 = L_pullback + (H - L_pullback) × 50%
```

```text
收盘从下向上突破回调波段50%突破位 → 右侧买入
收盘重新跌破该批次买入结构 → 止损
```

如果出现低于 `L_pullback` 的新低，旧突破位失效，必须使用新低重算。没有出现新低时，止损后股价再次满足同一位置的有效突破条件，允许在同一个买入卖出点重新交易；每一次都作为独立批次重新绑定止损。

普通波段比例位的“拉回买入”和回调波段50%突破位的“向上突破买入”是两种不同操作，不得混淆。

## 62.5%与75%

- 75%代表较浅回调、62.5%次之、50%更深；比例位仍需实际技术支撑共振。
- 某级别上涨波段62.5%未被跌破时，不使用回调波段一半突破；这种浅回调更可能形成反复突破、跌破的平台，应等待21日以上收盘前高突破等其他买点。
- 波浪级别必须与计划持有周期匹配；过大级别只说明长期方向，不能直接充当短期买点。
- 原波浪因重叠、新低或结构破坏失效时，必须重新划分。

## 左转右与批次独立

原左侧底仓与后续右侧加仓，先视为不同交易：

```text
底仓：使用原估值或长期支撑止损
加仓1：使用突破位止损
加仓2：使用量价节点止损
```

右侧加仓失败时只退出加仓，不因为平均成本较低而放宽新仓止损。新增仓位取得10%至20%浮盈后，才可与底仓合并进入统一分仓止盈。

## 三类止损

### 条件止损

收盘跌破买入依据的上涨波段50%支撑位、回调波段50%突破位、量价节点、均线、缺口或颈线。明确规定盘中止损的结构按盘中处理。

### 幅度止损

```text
幅度止损价 = 成本 × (1 - 5%～10%)
```

比例在买入前确定，10%通常是极限，触发后不临时放宽。

### 时间止损

右侧买入后5至13个交易日，最大浮盈仍未达到约10%，说明资金效率或突破质量不足。

```text
市场弱或Formula33下行：5日
中性阶段：8日左右
结构良好、无背离：13日
```

左侧仓默认不用时间止损。

## 浮盈加仓

- 首笔仓已经取得10%至20%浮盈。
- 后续再次出现独立的突破或拉回买点。
- 单次新增仓位不超过已有仓位的1/2；突破新高的不确定买点可降至1/4至1/3。
- 新仓绑定新买点止损，不能与底仓混合。
- 新仓取得安全垫后，才与底仓合并。

## 分仓止盈

### 启动条件

最大浮盈10%以内不止盈，只执行条件、幅度和时间止损。最大浮盈10%至20%之间，只启用“最大浮盈回撤一半”并卖出一份。最大浮盈超过20%后才启动多条件分仓止盈。

分仓通常使用五份，也可以按实际有效条件使用四至七份。份数由互不重复的技术卖点决定，不能为了凑数随意设置条件。

### 常见三份

1. 时间不过高：无背离给8至13日；顶背离后缩短为5日。
2. 关键结构：量价节点、连阳一半、趋势线、均线、颈线等。
3. 最大浮盈回撤一半。

```text
最大浮盈回撤一半 = 成本 + (最高价 - 成本) × 50%
```

### 扩展五份

4. 下一级有效量价节点或结构支撑。
5. 再下一级节点、中长期支撑或最终趋势保护。

成本上方5%、峰值回撤10%也可以分配给某一份，但不能默认作用于整仓。

```text
成本上方5% = 成本 × 1.05
峰值回撤10% = 最高价 × 0.90
```

### 重置条件

- 再次创新高：不过高计时归零。
- 当前量恢复到上涨趋势基准量以上：背离后的缩量计时重新评估。
- KD/RSI背离消失或转为钝化：重新评估对应时间份额。

连续涨停期间每日创新高，且没有跌破结构、没有有效背离时，不应机械止盈全部仓位。这是多氟多能够持有连续涨停的重要原因。

## 均线均量扣抵的作用

- 上扬中长期均线附近，价格在线上是潜在支撑，在线下是向上拖拽候选。
- 下弯均线位于上方，量不足时形成压力和向下拖拽。
- 均量扣低意味着较小增量即可推动均量线转强；扣高量时需要更大量能克服压力。
- 上涨趋势基准量可参考5日、10日均量扣抵量水平。
- 指数和板块优先使用成交金额；个股可结合成交量和成交额。

## 案例形成的程序约束

### 均胜电子

- 同时具备估值条件时可做左侧五份。
- 放量收复结构后左转右，但新增仓必须独立止损。
- 单日放量不等于右侧确认，仍需MA20、波段或结构条件。

### 多氟多

- 财务价值线远低于当时市场价，近期行情没有触发左侧；若未来进入经复核的低估区，仍允许按预设网格做左侧。
- 低位支撑和波段买点提前建立成本优势，连续创新高会重置时间止盈。
- 盈利保护只能绑定某一份，不能因浮盈达到10%就整仓清仓。

### 天赐材料

- 上涨波段50%支撑位约38.4是当前拉回支撑候选。
- 上涨波段50%支撑位失守后不向下补仓，等待新低形成并重算回调波段50%突破位。

### 瑞芯微

- 市场价远高于当时财务价值线，近期行情没有触发左侧；左侧资格由估值和价格位置动态决定，不做永久排除。

# 日 K 条件单撮合

- 预挂突破、网格和硬止损订单使用当日开盘价、最高价、最低价判断成交，不使用事后收盘价代替盘中成交价。
- 跳空穿越订单价格时按开盘价成交；突破买入高开超过触发价 5% 时取消追入。
- 仅触及订单价但没有穿越时，日 K 无法确认排队成交，回测按未成交处理。
- 一字涨停默认买单不能成交，一字跌停默认卖单和止损单不能成交；反方向订单可按价格条件成交。
- ST 股票在 2026-07-06 前按 5% 涨跌幅限制识别，自 2026-07-06 起按 10% 识别。
- 同一根日 K 同时覆盖多个订单价格且无法判断先后时，采用保守成交顺序，不选择对收益更有利的路径。
- 开第二、第三只右侧股票所需的浮盈门槛只使用此前交易日已经确认的数据；不使用当日收盘价反推盘中更早时刻具备开仓资格。
- 普通A股新买批次遵守T+1：当日收盘跌回突破结构只能标记待确认，次日才能卖出；卖出昨日批次后，当天再次突破允许按新批次买回，新批次重新进入T+1。
